What Does It Do?

This library is an add-on for SuperScript. It is strongly recommended to read the documentation for the core project before proceeding with this library. If you haven't yet read the core documentation then it is recommended that you start by reading about the core concepts.

SuperScript.Templates.JavaScriptTemplates adds functionality for declaring HTML templates, written for the JavaScript Templates library in server-side .NET code.

While the SuperScript library contains the core functionality for making and emitting declarations, SuperScript.Templates.JavaScriptTemplates contains a specific implementation of one of those core classes. Broadly speaking, SuperScript has three main aspects: declarations, which are instances of DeclarationBase, and emitters, which consist of a combination of modifiers.

SuperScript.Templates.JavaScriptTemplates therefore consists an implementation of a specific modifier, an CollectionConverter. In fact, this implementation, JavaScriptTemplatesStringify, is part of a hierarchical chain:

  • JavaScriptTemplatesStringify
  • ↓
  • TemplateStringify
  • ↓
  • CollectionConverter
  • ↓
  • ModifierBase
